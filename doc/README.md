# API Documentation

This folder contains documentation for all backend APIs, including request and response formats, to help frontend integration.

- `member-api.md`: Member management endpoints
- `plan-api.md`: Plan management endpoints
- `membership-api.md`: Membership endpoints
- `document-api.md`: Member document upload endpoints
- `misc-api.md`: Miscellaneous endpoints

Each file details:

- Endpoint URL
- HTTP method
- Request parameters/body
- Response format
- Error responses

---

Refer to individual files for details on each API group.
